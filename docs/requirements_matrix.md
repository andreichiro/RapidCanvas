# Gate 1 Requirement Matrix

This matrix is the assignment closure gate. Every source requirement is mapped to implementation files, tests or review gates, eval artifacts, documentation, and current status. Gate 1 is accepted only when `make requirements-review` passes with zero missing mappings.

Statuses:
- `implemented`: present in the scaffold and validated now.
- `planned`: explicitly owned by a later gate with implementation, tests, eval, and docs mapped.
- `reserved`: intentionally scaffolded now and completed by a named later gate.
